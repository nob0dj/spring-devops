* devtools
* lombok
* web
* jpa
* oracle
* redis
* session

- oracle cloud 지갑 관리 (submodule)
- secret 환경변수 관리
- 배포환경 redis-docker 연동
- github action
  - aws ec2 cicd
  - docker build
  - 환경변수 처리
  - aws security group ssh 접속 ip 일시 추가 및 제거
